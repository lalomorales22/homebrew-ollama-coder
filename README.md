# homebrew-ollama-coder

Homebrew tap for [ollama-coder](https://github.com/lalomorales22/ollama-coder) - an agentic coding assistant for Ollama (like Claude Code, but local!)

## Installation

```bash
brew tap lalomorales22/ollama-coder
brew install ollama-coder
```

## Usage

```bash
ollama-coder
```

## Requirements

- [Ollama](https://ollama.ai) must be installed and running
- At least one model pulled (e.g., `ollama pull llama3.2`)

## Upgrading

```bash
brew upgrade ollama-coder
```

## Uninstalling

```bash
brew uninstall ollama-coder
brew untap lalomorales22/ollama-coder
```
